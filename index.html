자율 코딩 에이전트를 위한 GitHub 중심 컨베이어 벨트

GitHub = 영속 상태 머신 이슈 → 빌드 큐 PR → 리뷰 큐 라벨 → 상태 CI → 품질 게이트

에이전트(claude / codex)는 헤드리스 모드의 stateless 워커 — 격리된 git worktree에서 한 번에 하나의 작업만 수행합니다.
리뷰가 막히면 중재자(Arbiter)가 최종 결정(SHIP / REDO)을 내려 막다른 길을 없앱니다 (agent:blocked만 드문 예외).
